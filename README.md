## What is this?

A small Python script that draws **RDJ (Robert Downey Jr.)** using code.

Yes, really.
No, it’s not magic.
(Okay, maybe a little.)

---

## Features

* Draws RDJ like a boss
* Uses Python (because of course it does)
* Makes you feel like Tony Stark for ~10 seconds

---

## Usage

Run it:

```bash
python RDJ_Draw.py
```

Then sit back and watch:

> "Sometimes you gotta run before you can walk."

Install dependencies:

```bash
pip install -r requirements.txt
```
